# Gesture & Voice Controlled Virtual Mouse

An intelligent Human-Computer Interaction (HCI) system that allows users to control a computer using **hand gestures**, **voice commands**, and a **web-based interface**, eliminating the need for traditional input devices like a mouse and keyboard.

---

## 🚀 Features Overview

The system is divided into three major modules:

- ✋ Gesture Control Module  
- 🎙️ Voice Assistant Module  
- 🌐 Web Interface Module  

---

## ✋ Gesture-Based Controls

### 🖱️ Cursor & Navigation
- Move Cursor using fingertip tracking  
- Neutral Gesture to pause/stop control  

### 🖱️ Click Actions
- Left Click  
- Right Click  
- Double Click  

### 🖱️ Advanced Operations
- Drag and Drop  
- Multiple Item Selection  

### 🔄 Scrolling
- Vertical Scrolling  
- Horizontal Scrolling  
- Speed varies based on gesture movement  

### 🔊 System Controls
- Volume Control using pinch gesture  
- Brightness Control using dynamic gestures  

---

## 🎙️ Voice Assistant (Proton)

### ⚙️ System Control
- `Proton Launch Gesture Recognition` → Start gesture system  
- `Proton Stop Gesture Recognition` → Stop gesture system  

### 🌐 Web Commands
- `Proton search <query>` → Search on Google  
- `Proton find a location` → Open location in Maps  

### 📁 File Navigation
- `Proton list files` → List files in directory  
- `Proton open <number>` → Open file/folder  
- `Proton back` → Go to parent directory  

### ⏰ Utility
- `Proton time` → Get current time  
- `Proton date` → Get current date  

### 📋 Productivity
- `Proton copy` → Copy selected text  
- `Proton paste` → Paste copied text  

### 💤 Assistant Controls
- `Proton bye` → Sleep mode  
- `Proton wake up` → Resume  
- `Proton exit` → Exit assistant

## 🧠 Tech Stack

- Computer Vision: MediaPipe, OpenCV  
- Machine Learning: CNN  
- Voice Processing: Speech Recognition, PyAudio  
- Backend: Python  
- Frontend: ReactJS

---

## ⚙️ Working

1. Webcam captures hand movements  
2. MediaPipe detects hand landmarks  
3. Gestures are classified and mapped to actions  
4. Voice assistant processes commands  
5. Web interface manages and monitors the system
